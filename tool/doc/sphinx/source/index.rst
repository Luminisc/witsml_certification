About
-----

The WITSML Test Tool was developed to fulfill the following objectives:

- Allow functional testing of a WITSML server

- Enable testing GetFromStore-only servers

- Simplify the preparation and maintenance of test scripts

Test Tool Packages
------------------

Certification tool consists of several packages 


**wtl** - WITSML Test Tool

**wsvt** - WITSML Schema Validation Tool

**wtools** - WITSML dataset tools.

How to write this documentation
-------------------------------

This documentation is generated by Sphinx 

You can install Sphinx by typing 

*easy_install sphinx*

if you do not have *setup tools* in your path, or if you not using state of art operation system like Linux you can download windows binary:

`Sphinx Binary <http://www.lfd.uci.edu/~gohlke/pythonlibs/#sphinx>`_  

make sure sphinx is in your system path, you won't have problem on Linux, but on Windows make sure that c:/Python27/Scripts is in your path

The best way to jump start is to use Eclipse plugin for sphinx, then you don't need to deal with command line you can download ecplise plugin from :

*http://resteditor.sourceforge.net/eclipse* 

then in eclipse you click on index.rst file and click run, this will run HTML documentation.
