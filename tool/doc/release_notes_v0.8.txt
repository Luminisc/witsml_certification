Release Notes - Version 0.8

1.    Description

This is the first public release of the WITSML Test Library (WTL)
It contains the basic functionality to develop and run Test Script and Test
Suites against a WITSMl server

2. Documentation

The following documentation is provided with this release:
  
  - Read me file: "README"
  - WITSML Test Tool User Guide: "WITSML Certification Tool Documentation.docx"
  - Sample script description: "sample_script_summary.txt" 
  - Release Notes: This document 

3.    Code Status

This is the first release of this code: 
- Only a few primitives have been tested
- Some primitives have not been tested at all
- Additional primitives still have to be implemented

It is expected that the development of this tool with proceed together with
the development of the WITSML certification test plan. 
The initial set of primitives were designed based on the expected required
functionality. Once the test plan is completed the actual required set of
primitives can be completed

List of things still to be done is included in the initial version of the
user guide provided
